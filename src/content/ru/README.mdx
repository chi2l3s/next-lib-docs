# NextLib

NextLib — модульная библиотека для Paper/Spigot плагинов.

Эта документация построена по исходникам `chi2l3s/next-lib` и отражает текущее runtime-поведение API.

## Модули

- Dynamic Database
- GUI API
- Command API
- Item API
- Color API
- Config Manager
